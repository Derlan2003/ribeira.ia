# Apresento CRM — Landing Page

Landing page do **Apresento CRM**, desenvolvida por [RIBEIRA.IA](https://ribeira.ia).

## Estrutura do Projeto

```
apresento-crm/
├── index.html              # Página principal
├── assets/
│   ├── css/
│   │   └── style.css       # Estilos globais
│   ├── js/
│   │   └── main.js         # Scripts e módulos
│   └── images/
│       └── hero-image.jpg  # Imagens (adicionar aqui)
└── README.md
```

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/apresento-crm.git
   ```
2. Abra o `index.html` no navegador ou suba para o GitHub Pages.

## Hospedagem no GitHub Pages

1. Vá em **Settings → Pages** no repositório
2. Em **Source**, selecione a branch `main` e pasta `/ (root)`
3. Salve — o site estará disponível em `https://seu-usuario.github.io/apresento-crm`
